# Theramotion Web App

Frontend starter scaffold using:

- React + Vite
- Tailwind CSS
- Zustand
- Lucide React

## Folder structure

Under `/src`, the project includes empty starter folders:

- `views/`
- `components/`
- `store/`
- `hook/`
- `util/`

## Scripts

- `npm run dev` — start development server
- `npm run build` — create production build
- `npm run lint` — run ESLint
